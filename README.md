# Stroke-Risk-Analysis
Stroke Risk Analysis Dashboard using Excel & Power BI
Project Overview

This project focuses on analyzing healthcare data to identify factors associated with stroke occurrence. Using Excel and Power BI, the dataset was cleaned, transformed, and visualized to generate meaningful healthcare insights through an interactive dashboard.

The project analyzes patient demographics, medical conditions, and lifestyle factors such as hypertension, heart disease, smoking status, BMI, and glucose levels to understand stroke patterns.

Tools & Technologies Used

Excel
-  Data Cleaning
-  Pivot Tables
-  Conditional Formatting
-  Basic Analysis

Power BI
-  Power Query
-  Data Visualization
-  Dashboard Creation
-  DAX Measures
- Interactive Slicers

Data Cleaning Process
 The following preprocessing steps were performed:

 - Removed/handled missing BMI values
 - Checked for duplicate records
 - Converted categorical fields for analysis
 - Created Age Groups
 - Created BMI Categories
 - Standardized column formats

   Key Insights
Stroke occurrence was higher among older patients.
Patients with hypertension showed higher stroke risk.
Smoking habits influenced stroke occurrence.
Higher glucose levels and BMI were commonly observed among stroke patients.
Female patients recorded slightly higher stroke cases in the dataset.

Conclusion
  This project demonstrates how healthcare data can be transformed into actionable insights using Excel and Power BI. The dashboard helps visualize stroke-related risk factors and supports better understanding of patient health patterns.
